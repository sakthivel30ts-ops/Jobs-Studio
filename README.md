# Jobs Studio - Job Portal Application

A full-stack Job Portal built using Spring Boot and React.js, designed to connect job seekers with employers efficiently.

## Features

### Backend (Spring Boot)
- Complete RESTful API development with Spring Boot
- User authentication and authorization (Job Seeker & Employer roles)
- Job posting, job search, and application management
- Advanced filtering options (Location, Experience, Salary, Job Type)
- Proper exception handling and validation
- MySQL database with optimized queries

### Tech Stack
- **Backend**: Java, Spring Boot, Spring Data JPA, Spring Security
- **Database**: MySQL
- **Frontend**: React.js (separate repository)
- **Tools**: Maven, Postman, Git

## Project Structure
Implemented layered architecture with Controller, Service, Repository pattern for clean and maintainable code.

This project demonstrates my ability to build scalable backend systems using Spring Boot and design proper database architecture.
